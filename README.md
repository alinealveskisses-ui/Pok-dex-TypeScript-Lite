# Pokédex TypeScript Lite

## Sobre o projeto
O Pokédex TypeScript Lite é uma aplicação utilitária desenvolvida em Node.js com TypeScript puro que realiza a consulta de dados de Pokémon através da PokeAPI e gerencia um catálogo local armazenado estritamente em memória durante a execução do programa.

## Objetivo
Praticar os principais conceitos de desenvolvimento back-end:
- JavaScript/TypeScript no ambiente Node.js.
- Consumo de API assíncrona usando `fetch` e controle com `async/await` e `try/catch`.
- Programação Orientada a Objetos (Classes, atributos privados, métodos e modificadores de acesso).
- Manipulação avançada de estruturas com métodos funcionais de array (`map`, `filter`, `some`, `forEach`).
- Versionamento semântico de código com Git/GitHub seguindo fluxo GitFlow.

## Tecnologias utilizadas
- Node.js (v20+)
- TypeScript (Strict Mode)
- TSX (Executor de desenvolvimento)
- PokeAPI (API pública externa)

## Pré-requisitos
Antes de executar o projeto, é necessário ter instalado:
- Node.js
- npm
- Git
